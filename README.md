# Spectre v5.0

> Network diagnostics and security research framework for authorized testing environments.

---

## Requirements

### Linux
```bash
sudo apt update && sudo apt install python3 python3-pip
pip3 install colorama requests
```

### Windows
```cmd
python -m pip install colorama requests
```

### Optional
```bash
pip install pyzipper pillow pynput
```

---

## Installation
```bash
git clone https://github.com/CoSpiky/spectre.git
cd spectre
python3 Spectre.py
```

---

## Usage

Navigate pages with **N** (next) and **B** (back). Type a tool number to view its description, then press **Enter** to launch. Type **Q** to quit.

---

## Features

- Network scanning and diagnostics
- OSINT gathering and data analysis
- Hash generation and verification
- System information and diagnostics
- Custom payload generation for authorized testing
- API interaction and automation utilities
